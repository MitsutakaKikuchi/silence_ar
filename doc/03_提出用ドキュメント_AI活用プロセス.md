

作品名: 『沈黙の解剖図譜 —— 不可視の愛に関する空間的考察』

報告者: 菊池光峰(2125173) 所属:音楽学部邦楽科 長唄三味線専攻
使用モデル: Gemini 3 Pro/Flush (企画・設計・実装・執筆補助), Gemini Nano Banana Pro (ビジュアル生成) 

1. 制作意図と背景：感情の「言語化」から「実装」まで
1.1 コンセプト：沈黙の可視化
本作は、人間関係における「言わなかった言葉（沈黙）」が持つ二面性 -相手を守るための「救い」と、自身を蝕む「呪い」-を視覚化する試みである。「言わないからこその優しさ」「言えなかった苦しみ」
目に見えない感情の澱（おり）を、AIという「客観的な他者」の視点を介することで「解剖図（アナトミー）」として再構築し、鑑賞者が自身の記憶を投影できるWebAR体験として構成した。
1.2 生成AIによるフルスタック制作体制
本プロジェクトの特異性は、構想・素材生成・実装・体験設計に至る全工程を、GoogleのAIモデル（Gemini family）との協働により完遂した点にある。私個人としての制作能力をAIによって拡張し、企画からデプロイまでを短期間でプロダクトレベルへと引き上げることができた。




















2. 概念と言葉の構築 (Gemini 3 Pro)
2.1 抽象概念のメタファー変換
「優しさが逆に苦しい」といった矛盾する感情を、一貫して「庭にまつわるオブジェクト（植物、土、園芸具）」に置き換えた。これにより、抽象的な感情に物質的な実在感を持たせている。

入力例: 「厳しい言葉で正される感覚」
Gemini 3 Proの提案: 「植物の自由を奪いながらも、倒れないように支える鉄の支柱」
実装: このメタファーを元にエピソード・テキストを10項目作成した。






2.2 ARレイヤー構造の設計
AIとの対話から、「物理的な日常（Target Image）」と「デジタルの本音（AR Overlay）」を重ね合わせる構造を採用した。

Layer A (Target): 日常のスナップショット。
Layer B (Content): AI生成による概念的解剖図。
テキスト: 憂鬱だが理知的で詩的な語り口を採用し、2文構成のポラロイドキャプション形式とした。


3. ビジュアル生成プロセス (Gemini Nano Banana Pro)
3.1 意図的な画風の対比
Gemini Nano Banana Proを用い、あえて全く異なる2つのスタイルを出力し分けることで、ARスキャン時の「真実が露わになる感覚」を強調した。
日常の記録 (Daily Life / Layer A): 「スマートフォンによる撮影」「不完全なフォーカス」等のプロンプトを用い、リアリティのあるスナップ写真を生成。チェキ画像風に整え、印刷。
真実の解剖 (Inner Truth / Layer B): 「緻密な植物細密画」「シュールレアリスム」「明暗法」をキーワードに、幻想的かつ不穏な高精細ビジュアルを生成。

4. WebAR実装と技術設計
4.1 技術スタックと構築
Gemini 3 Proの提案に基づき、アプリ不要で体験可能な「MindAR」と「A-Frame」を採用した。コードの実装においても、AIを活用してイベントハンドリングやモバイル最適化を行っている。














4.2 インスタレーションアートとしての演出
デジタル作品としての「静止」を避け、常に変化し続ける環境をJavaScriptで実装した。

時間帯連動型演出: 閲覧時の時刻を取得し、朝・昼・夕・夜でテーマカラー（CSS変数）を自動変更するシステムを構築。
パーティクルシステム: Canvas 2D APIを用い、花びらや光の粒子が浮遊する有機的な空間を演出。
没入型UX:
浸透的ローディング: 4段階のシーケンスを経て、徐々に作品世界へ引き込む導入部。
シアターモード: 詩のフラグメントが浮遊し、アンビエントサウンド（Web Audio APIによる
動的合成）が流れるモードの実装。

4.3 解剖学的露見シェーダー (Anatomical Reveal Shader)
インスタレーションとしての「独創性」を最大化するため、単なる画像の切り替えにとどまらず、作品のコンセプトである「解剖（Anatomy）」と「沈黙の有機的な蠢き」を視覚化する高度なシェーダーを実装した。

概念: 「表層（日常/Layer A）が、内圧（本音/Layer B）によって有機的に浸食され、裂け目から深淵が覗く」という表現を、Three.jsのカスタムシェーダーで実現。

技術的特徴:
視線追従パララックス: 端末の角度（ジャイロセンサー）に応じて、Depth Mapに基づいた「穴の奥」が動いて見えるだまし絵的な立体感を実装。これにより、2D画像が奥行きを持った「解剖図」として知覚される。
有機的ノイズ浸食: シンプレックスノイズを用いて、境界線が直線ではなく、細胞や黴（かび）のように有機的なノイズを持って変容する演出。日常と本音の境界が「生きている」かのように蠢く。
色収差 (Chromatic Aberration): 境界線にデジタルな「痛み」や「ノイズ」を表す色ズレを発生させ、現実とデジタルの狭間を視覚的に強調。
角丸フェードマスク (Rounded Corner Mask): ターゲットエンティティの境界を角丸処理し、エッジにフェード効果を適用。物理的な写真が浮かび上がるような柔らかい表示を実現。

演出意図: スキャン直後は「日常（Layer A）」のみが表示されるが、ターゲットを認識するとAR上の画像が有機的に裂けて「本音（Layer B）」が露出し、端末を傾けることで奥行き（Depth）を感じられる。これにより、「鑑賞者の行為（スキャン）によって、日常の皮膜が剥がれ落ちる」という動的で哲学的な体験へと昇華される。

4.4 パフォーマンス最適化
WebARにおいてモバイル端末での スムーズな体験を担保するため、以下の最適化を施した。

テクスチャキャッシュ: 同一画像（target_source.jpg等）の重複読み込みを防止するグローバルキャッシュを実装。初回読み込み後はキャッシュから即座に参照され、10ターゲット間での共有が効率化される。
画像圧縮: メインターゲット画像を品質を維持しつつ約88%削減（4.8MB→0.6MB）。初期読み込み時間を大幅に短縮。























5. 結論と考察
本プロジェクトを通じ、AIは単なる「素材生成ツール」ではなく、制作の全工程において論理的設計と感覚的表現を橋渡しする「クリエイティブパートナー」として機能した。
私自身の原体験から由来する多くの「沈黙」が、AIによる再解釈を経てWebARという普遍的な体験へと結実したことは、次世代の制作プロセスにおける一つの可能性を感じずにはいられない。
